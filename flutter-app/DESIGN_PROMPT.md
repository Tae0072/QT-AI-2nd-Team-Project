# QT-AI 디자인 프롬프트 — "Calm Paper" 스타일

> 기준 화면: 오늘 QT 탭 스크린샷 (2026-06-10).
> 용도: Flutter UI를 AI로 구현/수정할 때 이 문서의 "마스터 프롬프트"를 그대로 붙여넣는다.
> 모든 탭(오늘·성경·나눔·노트·마이)에 동일하게 적용한다.

---

## 1. 한 줄 컨셉

**"따뜻한 종이 위의 미니멀 묵상"** — 웜 오프화이트 배경, 잉크 같은 차콜 텍스트, 장식 없는 라인 아이콘, 콘텐츠가 주인공인 Apple풍 미니멀 디자인. 색은 거의 쓰지 않고, 위계는 명도·크기·여백으로만 만든다.

## 2. 디자인 토큰

### 색상 (Color)

| 토큰 | 값 | 용도 |
|---|---|---|
| `bg` | `#F7F5F2` | 화면 배경 (웜 오프화이트) |
| `surface` | `#FFFFFF` | 선택된 칩, 플로팅 시트, 카드 |
| `surfaceSunken` | `#F0EEEA` | 영어 번역·해설 등 보조 콘텐츠 박스 |
| `textPrimary` | `#1F1F1F` | 제목, 본문 (잉크 차콜) |
| `textSecondary` | `#8A8A8E` | 영어 번역, 절 번호, 보조 설명 |
| `textTertiary` | `#B8B5B0` | 영문 레퍼런스, 비활성 탭 라벨 |
| `hairline` | `#E8E6E2` | 칩 테두리, 구분선 |
| `accent` | `#E0492F` | 활성 탭 도트 등 극소량 포인트 (한 화면에 1곳 이하) |

규칙: accent 외 유채색 금지. 그라데이션 금지. 어두운 배경 블록 금지.

### 타이포그래피 (Typography)

| 토큰 | 크기/굵기 | 용도 |
|---|---|---|
| `titleXL` | 26px / w700 / -0.5 자간 | 화면 대제목 (예: "부끄러운 송사") |
| `titleBar` | 17px / w600 | 앱바 중앙 타이틀 (예: "오늘 QT") |
| `body` | 16px / w400 / line-height 1.65 | 한글 본문 (성경 구절, 노트 내용) |
| `bodySub` | 14px / w400 / 1.55 / textSecondary | 영어 번역, 인용 |
| `caption` | 13px / w400 / textSecondary | 해설 한 줄, 메타 정보 |
| `label` | 12px / w500 / textTertiary | 영문 레퍼런스, 탭 라벨 |

대제목 앞에는 주제 이모지 1개를 둘 수 있다 (예: ⚖️). 본문은 줄간격을 넉넉하게(1.6+) 유지한다.

### 간격·반경·그림자

- 화면 좌우 패딩: 20px. 섹션 간 세로 간격: 24px, 요소 간: 8~12px.
- 반경: 칩/필 = stadium(완전 라운드), 보조 박스 = 14px, 플로팅 시트 = 상단 24px.
- 그림자: 선택된 칩과 플로팅 시트에만 `0 2 8 rgba(0,0,0,0.06)` 수준의 아주 옅은 그림자. 그 외 그림자 금지.

## 3. 공통 컴포넌트 규칙

### 앱바
배경 투명(=bg), elevation 0, 타이틀 중앙 정렬. 우측에 라인 아이콘 최대 3개(테마 토글 ◐, 음악 ♪, 새로고침 ↻ 등). 아이콘은 textPrimary 단색 라인.

### 타이틀 블록
`이모지 + titleXL 제목` → 아래 한 줄에 한글 부제(caption, textSecondary) → 그 아래 영문 부제(label, textTertiary). 좌측 정렬.

### 세그먼트 칩 (해설/시뮬레이터/노트/영어)
- 가로 스크롤 가능한 필 칩 행.
- 선택: `surface` 배경 + hairline 테두리 + 옅은 그림자 + textPrimary w600.
- 비선택: 배경 투명(또는 bg) + hairline 테두리 + textSecondary.
- 채워진 유채색 칩 금지.

### 콘텐츠 리스트 아이템 (구절 패턴)
1. 절 번호: caption 크기, textSecondary, 본문 좌측에 고정.
2. 한글 본문: body, textPrimary.
3. 보조 블록: `surfaceSunken` 배경, 반경 14px, 내부 패딩 14px — 영어 번역(bodySub) + 해설 한 줄(caption)을 담는다.
4. 아이템 사이는 구분선 없이 24px 여백으로만 구분.

### 플로팅 바텀 시트 ("오늘의 묵상 질문" 패턴)
- 탭바 위에 떠 있는 `surface` 시트, 상단만 24px 라운드, 옅은 그림자.
- 헤더: caption 제목 + 우측 X 닫기 버튼.
- 내용: `surfaceSunken` 박스 안에 body 텍스트.
- 다른 탭에서도 "한 줄 요약/질문/안내"가 필요하면 이 패턴을 재사용한다.

### 하단 탭바
- `surface` 또는 bg 배경, 상단 hairline 1px.
- 탭 5개: 오늘(☀)·성경(⊞)·나눔(♡)·노트(≡)·마이(👤) — 모두 라인 아이콘.
- 활성: 아이콘+라벨 textPrimary, 아이콘 위(또는 옆)에 `accent` 도트 1개.
- 비활성: textTertiary. 채움(filled) 아이콘 전환 금지.

## 4. 탭별 적용 가이드

모든 탭은 §2 토큰 + §3 컴포넌트만 사용한다. 새 색·새 반경·새 그림자를 만들지 않는다.

- **오늘**: 기준 화면 그대로. 타이틀 블록 + 칩 + 구절 리스트 + 묵상 질문 시트.
- **성경**: 타이틀 블록("성경" + 책/장 부제) + 책 선택은 필 칩 또는 surfaceSunken 리스트 행. 장 그리드는 stadium이 아닌 14px 라운드 사각 셀, hairline 테두리.
- **나눔**: 게시글 카드는 카드 테두리 대신 24px 여백 + hairline 구분 또는 surfaceSunken 박스. 작성자/시간은 caption, 본문은 body. 좋아요·댓글 아이콘은 라인 아이콘 + textSecondary.
- **노트**: 날짜 헤더는 label, 노트 항목은 구절 패턴과 동일한 리듬(제목 body w600 + 미리보기 caption). 작성 FAB 대신 가능하면 상단 우측 라인 아이콘 사용.
- **마이**: 프로필 영역은 타이틀 블록 변형(이모지 대신 아바타). 설정 항목은 surfaceSunken 그룹 박스(14px) 안의 행 리스트, 행 사이 hairline.

## 5. 마스터 프롬프트 (복붙용)

```
다음 디자인 시스템을 정확히 따라 QT-AI Flutter 화면을 구현해줘.

[컨셉] 따뜻한 종이 질감의 Apple풍 미니멀. 콘텐츠 중심, 장식 배제.

[색] 배경 #F7F5F2, 카드/선택칩/시트 #FFFFFF, 보조박스 #F0EEEA,
주텍스트 #1F1F1F, 보조텍스트 #8A8A8E, 3차텍스트 #B8B5B0,
헤어라인 #E8E6E2, 포인트(활성 탭 도트 전용) #E0492F.
이 8색 외 유채색·그라데이션·어두운 블록 금지.

[타이포] 대제목 26 w700(앞에 주제 이모지 1개 가능), 앱바 타이틀 17 w600 중앙,
본문 16 w400 줄간격 1.65, 영어/인용 14 보조색, 캡션 13 보조색, 라벨 12 3차색.

[레이아웃] 좌우 패딩 20, 섹션 간 24, 요소 간 8~12.
구분선 대신 여백으로 위계를 만든다.

[컴포넌트]
- 앱바: 투명 배경, elevation 0, 우측 라인 아이콘 최대 3개.
- 세그먼트 칩: stadium 필. 선택=흰 배경+헤어라인 테두리+옅은 그림자+w600,
  비선택=투명+헤어라인+보조색. 유채색 채움 금지.
- 보조 콘텐츠(번역·해설·미리보기): #F0EEEA 박스, 반경 14, 패딩 14.
- 플로팅 시트: 탭바 위 흰 시트, 상단 24 라운드, 캡션 제목+X 버튼, 내용은 보조박스.
- 탭바: 라인 아이콘 5개(오늘·성경·나눔·노트·마이), 상단 헤어라인 1px,
  활성=주텍스트색+빨간 도트 1개, 비활성=3차색. filled 아이콘 전환 금지.

[그림자] 선택 칩·플로팅 시트에만 0 2 8 rgba(0,0,0,0.06). 그 외 금지.

이제 [화면 이름] 화면을 위 규칙으로 만들어줘: (화면 요구사항 작성)
```

## 6. 기존 코드 토큰 매핑

현재 `lib/core/theme/app_theme.dart`는 세이지 그린 웜 파스텔 테마다. 이 디자인으로 전환 시:

| 기존 | 변경 |
|---|---|
| `AppTheme.bg #FBF7F0` | `#F7F5F2` |
| `AppTheme.bgSunken #F4ECDF` | `#F0EEEA` |
| `AppTheme.bgElevated #FFFDF8` | `#FFFFFF` |
| `AppTheme.accent #7C9A7E` (세이지) | `#E0492F` (도트 전용으로 축소) |
| `AppTheme.text #4A443C` (브라운) | `#1F1F1F` (차콜) |
| `AppTheme.text2 / textMuted` | `#8A8A8E` / `#B8B5B0` |
| `AppTheme.hairline #E8DDCA` | `#E8E6E2` |
| 카드 라운드 24 | 보조박스 14 / 시트 상단 24로 분리 |

간격·반경은 기존 `AppGap`/`AppRadius`/`AppPad` (`app_dimens.dart`) 토큰을 그대로 쓰되, `AppRadius.md(12)` 대신 14가 필요하면 `AppRadius.box = 14` 추가를 권장. 색 변경은 `app_theme.dart` 한 파일에서 끝낸다 — 화면 코드에 색 하드코딩 금지.
