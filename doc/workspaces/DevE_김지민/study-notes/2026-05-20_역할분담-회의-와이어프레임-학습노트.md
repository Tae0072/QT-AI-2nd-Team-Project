## 3. 로그인 경계 기준 (W1 Day3 핵심)

미인증 기능 화면 차단과 토큰 만료 이동 기준을 정리했다.

| 화면 유형                        | 로그인 전                 | 로그인 후                    |
| -------------------------------- | ------------------------- | ---------------------------- |
| 첫 진입                          | 로그인 화면               | Today QT 화면 (별도 홈 없음) |
| 기능 화면 (QT·성경·노트·나눔 등) | 로그인 화면으로 차단      | 정상 접근                    |
| 토큰 만료                        | 로그인 화면으로 자동 이동 | —                            |
| 비로그인 본문 열람               | 미지원                    | —                            |

| 항목               | 기준                                                                     |
| ------------------ | ------------------------------------------------------------------------ |
| 첫 진입            | 로그인 전 → 로그인 화면. 로그인 후 → Today QT 화면 (홈 화면 없음)        |
| 기능 화면 차단     | QT·성경·노트·나눔·마이페이지 등 전체 기능 화면은 로그인 후 접근          |
| 토큰 만료 처리     | Security Filter에서 토큰 만료 감지 → 로그인 화면 자동 이동 (이승욱 구현) |
| 비로그인 본문 열람 | 미지원 — 비로그인 상태에서 성경 본문, 해설 접근 불가                     |

> 로그인 경계 구현은 이승욱 담당 Security Filter에서 처리. 김지민은 본인 담당 화면(N·S)에서 `미로그인 진입 불가` 흐름이 와이어프레임에 반영되었는지, Flutter 연동 시 흐름이 올바르게 연결되는지 확인한다.

## 6. 김지민 담당 범위 최종 확인

### 6-1. 백엔드 API

| API            | 엔드포인트                                             |
| -------------- | ------------------------------------------------------ |
| 개인 노트 생성 | `POST /api/v1/notes` (PRAYER / REPENTANCE / GRATITUDE) |
| 나눔 공개      | `POST /api/v1/notes/{id}/share`                        |
| 나눔 목록      | `GET /api/v1/sharing-posts`                            |
| 나눔 상세      | `GET /api/v1/sharing-posts/{postId}`                   |
| 좋아요·취소    | `POST/DELETE /api/v1/sharing-posts/{postId}/like`      |
| 댓글 작성·조회 | `POST /api/v1/sharing-posts/{postId}/comments`         |
| 댓글 삭제      | `DELETE /api/v1/comments/{commentId}`                  |
| 공유글 삭제    | `DELETE /api/v1/sharing-posts/{postId}`                |
| 신고           | `POST /api/v1/reports`                                 |

### 6-2. Flutter 화면

| 코드 | 화면                    | 핵심 구현 포인트                       | 로그인 경계    |
| ---- | ----------------------- | -------------------------------------- | -------------- |
| N-01 | 노트 목록               | 달력↔목록 탭 전환, 연속기록일 표시     | 로그인 후 접근 |
| N-02 | 개인 노트 카테고리 선택 | 기도·회개·감사                         | 로그인 후 접근 |
| N-03 | 개인 노트 작성          | 1섹션, 저장 버튼만, TTS 미제공         | 로그인 후 접근 |
| N-04 | 노트 상세·수정          | 공유본 스냅샷 유지, 외부 공유 바텀시트 | 로그인 후 접근 |
| S-01 | 나눔 피드               | PUBLISHED + 페이지네이션               | 로그인 후 접근 |
| S-02 | 나눔 상세               | 좋아요·댓글·신고 UI                    | 로그인 후 접근 |
| S-03 | 나눔 작성               | confirmNicknamePublic 필수             | 로그인 후 접근 |
