## ⚠️ 회의 필수 합의 사항 (스프린트 기본 원칙)
- [ ] **기능 추가 금지:** 본 스프린트는 기존 기능 분담 및 구현에 집중하며, 새 아이디어는 시연 후 논의한다.
- [ ] **로그인 강제 원칙:** 모든 기능 화면은 로그인 후 접근하며, 비로그인 상태의 본문 열람은 지원하지 않는다.
- [ ] **저장 버튼 원칙:** 자동 저장 기능은 배제하며, 오직 임시저장(Draft) 및 저장(Saved) 버튼만 존재한다.
