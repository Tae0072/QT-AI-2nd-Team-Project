## 항목별 작업 요지

### ① MEDITATION 라벨 "묵상" → "QT" (`bugfix/note-labels-qt-record`)
- 라벨 출처 3곳만: `note_models.dart:9` `'MEDITATION':'묵상'→'QT'` · `post_category.dart:9` `'묵상'→'QT'` · `app_ko.arb`+`app_localizations_ko.dart` `catMeditation '묵상'→'QT'`.
- 일반 개념어는 건드리지 않음.

### ⑩ "노트" 명칭 충돌(하단 탭 vs 작성 버튼) (`bugfix/note-labels-qt-record`)
- 새 l10n 키 `navRecord="기록"`(ko/en) → `home_screen.dart:80` 탭 라벨 교체 · `noteListTitle`→"기록".
- `navNote="노트"`는 QT 화면 노트 버튼용으로 유지(탭과 더는 안 겹침).

### ④ 달력 카테고리 색점 + 표시기간 (`feature/note-calendar-display-qa`)
- `markerBuilder`로 그날 `CalendarDay.categories`별 색점(eventLoader 점1개 교체). 색매핑 5종(체크리스트와 통일)+점 상한. 날짜 탭 체크리스트는 유지.
- `calendarFormat` 상태화 + `onFormatChanged` + `availableCalendarFormats:{month:'한 달', twoWeeks:'2주'}` → ⑦과 통합.

### ⑥ 요일행 잘림 (`feature/note-calendar-display-qa`)
- `TableCalendar`에 `daysOfWeekHeight`(24~32) 설정(미설정 기본16이라 갤탭/폰트서 잘림). 필요시 `rowHeight`. **갤럭시탭 실기기 확인.**

### ⑦ "2weeks" 죽은 버튼 정상화 (`feature/note-calendar-display-qa`)
- 숨기지 않고 배선해 월↔2주 전환으로 살림(④ 토글과 동일 작업).

### ⑤⑧ 노트탭 통합 레이아웃 (`feature/note-tab-list-grouping`)
- 한 화면: 상단 캘린더(월/2주) + [카테고리칩(전체+5종) + 상태필터(전체/임시저장/저장)] + 목록.
- 목록↔달력 토글·별도 목록모드 제거(`noteCalendarViewProvider`+AppBar 토글 삭제).
- ⑤ 빠른작성 = 칩 컨텍스트 작성: 기도/회개/감사 칩 선택 시 "+ {카테고리} 작성" → N-02 생략 N-03 직행(category 고정). FAB 유지(전체→N-02). QT·설교 칩은 작성 진입 없음.

### ②⑪ 성경 노트버튼 직행 + 구절 첨부 (`feature/note-sermon-verse-args`)
- ⚠️ **선행(dev 기준): 이지윤이 dev의 `bible_browser_screen.dart`에 성경 조회 노트버튼 + `_openSermonNotes(range)` 핸들러를 먼저 추가**해야 함. dev엔 현재 "조회" 버튼만 있고 노트버튼/핸들러가 없음(20260604 분이 dev 미반영).
- **김지민**: `NoteEditArgs`에 `verseIds`(+라벨) 추가 → `NoteEditScreen`이 SERMON+verseIds 수신 시 본문 인용삽입 + 저장 시 첨부. repo는 이미 `verseIds` 수용.
- **이지윤**: 추가한 `_openSermonNotes(range)`를 `AppRouter.noteList` 대신 N-03 라우트(`AppRouter.noteEdit`)로 + `NoteEditArgs(category:'SERMON', verseIds: range.verses.map((v)=>v.id))`. `range.verses`에 `id` 이미 로드 → 좌표→id 변환 불필요.
- ⑪ 통일: N-03을 `NoteEditArgs(category 고정 [+verseIds])`로 직행하는 단일 진입 계약(QT 4섹션은 예외).

### ③⑨ 리치텍스트 소비 (`feature/note-richtext-integration`)
- 이지윤이 리치텍스트 전체(에디터+직렬화+렌더) 구현 → 김지민은 N-03/N-04 + 나눔 공유본에 "땡겨쓰기".
- 받을 계약 3종: ① 직렬화 포맷(body string) ② 에디터 위젯 ③ 렌더 위젯.
- ⑨(글자크기·들여쓰기)는 ③에 포함되어 해결. ⚠️ 빠른 QA 수정 아님 — 별도 피처 트랙.
