## 배경
- 백엔드는 김지민이 이미 완성: `GET /me/sharing-posts`(공개+숨김), `PATCH /sharing-posts/{id}/hide|show`, `DELETE /sharing-posts/{id}` (04 §4.4.5/4.4.6).

| 항목 | 결정 | 이유 |

| 액션 UI | **⋮ 오버플로우 메뉴** (인라인 버튼 X) | 관리 화면답고 카드 깔끔, 액션 늘어도 수용 |
