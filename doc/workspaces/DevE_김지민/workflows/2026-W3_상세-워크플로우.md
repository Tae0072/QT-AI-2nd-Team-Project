> 백엔드만 짜면 동작이 안 보이므로 **눈으로 확인**하는 단계를 매일 넣는다.
1. 앱 dev 프로파일 실행 (`spring.profiles.active=dev`)
2. 브라우저에서 `http://localhost:8080/dev-console.html` → `X-Dev-User-Id` 입력 → 그날 기능 버튼 클릭 → 응답 status·JSON 확인
3. `http://localhost:8080/h2-console` 접속 → 해당 테이블 행(스냅샷·likeCount·comment·report) 확인
   - JDBC: `jdbc:h2:mem:qtai;DB_CLOSE_DELAY=-1;DB_CLOSE_ON_EXIT=FALSE;MODE=MYSQL`, user `sa`, pw 빈값
   - ⚠️ 인메모리라 앱 재시작 시 데이터 초기화. 흰 화면이면 `DevSecurityConfig`에 `.headers(h -> h.frameOptions(f -> f.sameOrigin()))` 1줄 추가
- **자동 테스트(정확성) + 콘솔(눈으로 동작 체감)** 이중 확인.

- [ ] 중복 좋아요 409(DUPLICATE_LIKE) / 미존재 좋아요 취소 처리

**테스트 3종**
