> 이 핸드오프는 **2026-06-09 모놀리식 `qtai-server` 전제**. 2026-06-10 MSA가 **service-user / admin-server**로 더 분리되며 **F1 카카오(§6)·결정④** 부분이 바뀜. C·D3·E1·E2·E3는 유지.

- **응답 형태 5개 합의 완료(2026-06-10)**: 응답 키 `member`/`admin` 구분 · `adminRole` 단일 · refreshToken body · 403 ErrorCode 표시 · 만료 30분/14일.
- **✅ 엔드포인트 경로 확정(2026-06-10)**: 결정④ A(`/auth/kakao` 재사용) → 팀 결정으로 **신규 `admin/auth/kakao` 채택 확정**.

생성 `201` / 수정·발행 `200` / 숨김 `204` / 실패 `400·403·404·409·500`
