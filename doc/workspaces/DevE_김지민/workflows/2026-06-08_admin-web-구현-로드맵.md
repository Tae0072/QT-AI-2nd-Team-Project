## 1. 현재 상태 (골격 #318)

- **스택**: React 18 + TypeScript + Vite + Ant Design v5 + axios + React Router v6.
- **완비(재사용)**: `api/client.ts`(토큰 자동첨부·401 처리)·`unwrap<T>`, 공통타입 `ApiResponse/Page<T>/PageParams`,
  `AuthContext`+`ProtectedRoute`, `AdminLayout`(사이드바+헤더), `menu.ts`·`roles.ts`, api/*.ts 화면별 호출 stub.
- **비어있음**: 화면 실제 표/폼/모달, 공통 부품(목록 상태훅·토스트·상태배지), 세부 권한 메뉴 노출.
- **로그인**: 임시 토큰 직접 입력(localStorage). 카카오 웹 로그인 = 강태오. → **로컬 확인엔 유효한 ADMIN Bearer 토큰 필요**.
- **테스트**: 프레임워크 없음 → 게이트 = `npm run typecheck`(strict) + `npm run build`.
