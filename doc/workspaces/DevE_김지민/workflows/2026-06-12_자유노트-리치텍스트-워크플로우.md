## 변경 내용

### 1. 신규 공유 위젯 `widgets/note_rich_text_editor.dart`
- `NoteRichBodyController` — `buildTextSpan`을 파서로 오버라이드(라이브 프리뷰). QT/자유노트 공유.
- `NoteRichTextEditor` — 툴바 + @멘션 추천 + 본문 입력 한 묶음(ConsumerStatefulWidget). 포맷 핸들러(굵게/크기/색/들여쓰기/목록)·폰트/색 시트·@멘션 구절 삽입·자동 목록 접두사 전부 포함. `bodyLabel`/`bodyFieldKey`/`bodyScrollKey` 파라미터로 화면별 라벨·테스트 키 주입.

### 2. `screens/qt_note_editor_screen.dart` — 슬림화
- 편집 본체를 `NoteRichTextEditor`에 위임. passage 미리보기·제목·저장 버튼·`createQtNote` 저장만 유지. 공개 API(`QtNoteEditorScreen`/`QtNoteEditorArgs`) 불변 → today_qt/app_router 영향 없음.
- **이지윤 QT 테스트 11개 그대로 통과**(키·라벨·동작 보존: `qt-note-body-input`, `qt-note-mention-*`, `note-font-size-slider`, `note-color-swatch-*`, 툴팁, `[fs=]/[bg=]` 마커, 자동목록 등).

### 3. `screens/note_edit_screen.dart` (N-03) — 리치 에디터 적용
- 옛 마크다운 `NoteFormatToolbar` + 평문 TextField → `NoteRichBodyController` + `NoteRichTextEditor(bodyLabel: 본문)`. 저장 시 `_bodyController.text`(마커 포함 평문)를 그대로 `create`/`update`. 편집 모드 로드도 동일 컨트롤러에 채움.

### 4. N-04 상세(`note_detail_screen.dart`) — 변경 없음
- 이미 `QtNoteRichTextParser.parse`로 렌더 → 자유 노트의 새 마크업도 그대로 올바르게 표시됨.

### 5. 정리
- 옛 `widgets/note_format_toolbar.dart` 삭제(N-03 외 사용처 없음).
- 고아 l10n 키 `fmtBold/fmtItalic/fmtHeading/fmtList/fmtQuote/fmtCheckbox/fmtDivider` 제거 후 재생성.
