## 5. 아쉬운 점 / 개선 (Problem → Try)

- **PR 추적 혼선**: 어제(목) 저녁에 만든 PR을 todo엔 "금요일 예정"으로 적어둬, 금요일 아침에 PR 상태를 헷갈렸다. → **PR 생성 즉시 todo·워크플로우에 PR 링크/번호를 박자.** (gh 미인증이라 CLI로 확인 불가 → GitHub 웹/슬랙 봇 알림이 진실)
- **untracked 문서 방치**: 어제 만든 리포트 3개를 커밋 안 한 채 하루 넘김. → 마무리 routine 4단계(commit+push)를 그날 안에 닫기.
- **이지윤 CRUD 겹침**: B2에서 작업 영역이 겹쳐 merge abort. → 시작 전 dev 최신 + 담당 경계 재확인.
