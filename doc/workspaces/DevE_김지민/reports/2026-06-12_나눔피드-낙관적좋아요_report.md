## 한 일
- 카드 레이아웃은 이미 `PostCard`로 적용돼 있어, ③의 실제 작업은 **좋아요 낙관적 업데이트**였다.
- **피드(S-01)**: `sharingPostsProvider`를 `AsyncNotifier`(`SharingFeedNotifier`)로 전환 + `toggleLike(postId)` —
  즉시 로컬 갱신(낙관적) + 실패 롤백. 전체 재조회 없이 해당 글만 바뀐다. `PostCard` 하트 탭(onLike) + 실패 스낵바.
- **상세(S-02)**: `sharingPostDetailProvider`를 `AsyncNotifierProvider.family`(`SharingDetailNotifier`)로 전환 +
  `toggleLike()` — detail 좋아요만 즉시 갱신하고 **댓글은 재조회 없이 유지**, 실패 시 롤백. 화면 `_toggleLike`를
  `repo.like/unlike + _refresh()`(전체 재조회) → `notifier.toggleLike()`로 교체. 좋아요 후 피드는 백그라운드 동기화.
- 모델: `SharingPostItem.copyWith` / `SharingPostDetail.copyWith` 추가.
