- **응답 형태 5개 합의 완료(2026-06-10)**: ① 응답 키 `member`/`admin` 구분 유지 · ② `adminRole` 단일 문자열 · ③ refreshToken 앱·웹 모두 body(쿠키 X) · ④ 403 ErrorCode 그대로 표시 · ⑤ 만료 access 30분/refresh 14일.
- **✅ 엔드포인트 경로 확정(2026-06-10)**: 06-09 ③A(기존 `/auth/kakao` 재사용) → 2026-06-10 팀 결정으로 **신규 `POST /api/v1/admin/auth/kakao` 채택 확정**. 응답 형태 5개도 합의 완료.
- **미구현**: admin-server(8090)·`admin/auth/kakao`는 아직 없음(이승욱 task 1/3/4 대기). admin-web은 그때까지 dev 토큰 또는 기존 `/auth/kakao`로 로컬 확인.
