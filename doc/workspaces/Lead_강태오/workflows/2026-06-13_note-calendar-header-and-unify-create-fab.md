# 2026-06-13 기록 캘린더 헤더 우측 정렬 + 작성 FAB 형식 통일

## 배경(사용자 요청)
1. 캘린더의 오늘·week·다음달 버튼이 우측에 있어야 한다.
2. 기도/회개/감사 카테고리의 작성 버튼 형식(알약)이 전체 카테고리(둥근 +)와 달라 통일이 필요하다.

## 구현
- **캘린더 헤더**(`meditation_calendar.dart`): 기존 `[< 이전][년월][Spacer][오늘][week][> 다음]`에서, 년월 텍스트는 왼쪽에 두고 **오늘·week·이전·다음 버튼을 모두 오른쪽으로 묶었다**. 이전(<)·다음(>) 이동 버튼이 오른쪽에 나란히 위치.
- **작성 FAB 통일**(`note_list_screen.dart`): 기도/회개/감사 칩에서 뜨던 "○○ 작성" 알약을 제거하고, **모든 카테고리에서 전체와 동일한 둥근 액센트 + FAB**으로 통일. 동작은 유지 — 기도/회개/감사는 그 카테고리로 바로 작성(N-03), 그 외는 카테고리 선택(N-02). 미사용 색 상수(`_kFabBg`/`_kFabFg`) 제거.

## 검증
- `flutter analyze` → No issues.
- `flutter test` → **300개 전부 통과**(기도 칩 FAB 테스트를 둥근 FAB 기준으로 갱신).
