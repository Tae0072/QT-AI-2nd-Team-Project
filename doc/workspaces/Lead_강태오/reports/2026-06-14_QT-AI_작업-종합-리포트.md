작성: Lead 강태오 워크스페이스

범위: 모바일 앱(Flutter), 백엔드(qtai-server MSA), 로컬 운영 도구

## 1. 한눈에 보기

이 기간 동안 QT-AI(성경 QT·묵상 앱)의 **사용자 앱 기능 다수와 백엔드 보강**을 완료했다.
모든 코드 변경은 `dev` 대상 PR → CI 통과 → 머지 절차를 지켰고, 작성 시점 기준 **열린 PR 0개**다.

핵심 성과 영역:
- 홈/오늘의 말씀 랜딩, 노트 에디터(리치 텍스트 + 손그림), 나눔(저장·페이징·#멘션·댓글 신고)
- QT 본문 음성(TTS) 서버 캐시, QT 영상(시뮬레이터) 경로 정비
- 인트로(로딩) 연출, 기기 알림 브리지, 개발자 모드 테스트 허브
- 로컬 개발 환경 원클릭 부팅, DB 시드/인덱스/마이그레이션 보강

## 2. 영역별 결과 (대표 PR)

### 2-1. 로컬 개발 환경 / 운영 도구
- 원클릭 부트스트랩 `dev-up.ps1` + README/QUICKSTART를 MSA 구조로 정비 (#589)
- 전체 조회 속도 개선용 목록 인덱스 추가(V39) (#604)
- 로컬 DB 작업(백업, `bundle-all-books.sql` import, Flyway 클린 재시드)은 로컬 운영 작업이라
  PR 대상이 아니며, 재현 스크립트만 저장소에 보관(예: `scripts/dev-seed-qt-video.sql`).

### 2-2. 개발자 모드(내부 QA 도구)
- 설정 "버전 정보" 5탭 → 비밀번호 → 개발자 모드 진입 (#592)
- 전체 화면 허브: 모든 화면 바로가기 + 테마/폰트 상태 토글 (#623)
- 기기 알림 보내기 테스트 + 로컬 알림 서비스 (#600)
- 인트로(로딩) 화면 미리보기, QT영상 테스트 화면 (#631, #634)

### 2-3. 홈 / 오늘의 말씀
- 홈(랜딩) 화면 신설, 첫 탭을 홈으로 (#609)
- 오늘의 말씀 중앙정렬·세로 확장·이미지 배경, 매일 랜덤 자작 일러스트 (#610~#613)

### 2-4. QT 본문 / 해설
- 본문 장 전체 보기 + 선택 절 포커스 + 범위 선택→설교 노트 연동 (#565)
- 해설에 절 번호(장:절)와 절 본문 함께 표시, 굵기 정합 (#606, #607)
- 본문 노출 게이트(공개일) 강화 (#581, #584)

### 2-5. 노트(기록) 에디터
- 리치 텍스트 에디터를 `flutter_quill`로 교체 (#594)
- 서식 툴바(좌측 세로 패널), 저장 실패 수정 (#595)
- 손그림(펜)·원고/일반 페이지 모드, 지우개, 그림만 저장 (#596, #597)
- 기록 목록 다중 선택 삭제·빈 상태 안내·검색바·카드 재디자인 (#553, #563, #608)
- 캘린더 헤더 정렬·작성 FAB 정합 (#598, #601)

### 2-6. 나눔(커뮤니티)
- 게시글 저장(북마크) + 저장 목록 (#614), 게이트웨이 `/me/bookmarks` 라우팅 (#615)
- 피드 페이징(10개/페이지, 목차 5개씩) + 핫리로드 크래시 방지 (#616, #617)
- 닉네임 `#멘션` 기록·알림·내가 태그된 글 목록(백엔드/FE) (#618, #619, #620)
- 댓글 신고(프론트) (#621)
- 좋아요 낙관적 업데이트, 카드 디자인 개편 (#554, #560)

### 2-7. TTS(본문 음성)
- TTS 기본 서버를 배포된 Render 인스턴스로 (#624)
- 토큰 없이 재생 + 탭 시 생성 (#625)
- 오늘 QT 본문 음성 서버 지연 캐시(`qt_passage_audio`) + 앱 연동 (#626, #627)
- 로컬 `audio_data LONGBLOB` 명시로 "음성 생성 실패" 해결 (#629)
- 로딩 중 정지(취소) (#632)

### 2-8. QT 영상(시뮬레이터)
- READY 판정/캐시 보정, 버튼 스크롤 개선 (#569, #562)
- 개발자 모드 QT영상 테스트 화면 + 비활성 원인 진단 (#634)
- 샘플 영상 403 회피(Flutter 공식 영상으로 교체) (#636)
- 영상 어디든 탭하면 재생/정지 토글 (#637)

### 2-9. 인트로(로딩) 연출
- 빅뱅→T 십자가→8각별 시퀀스 (#630), 5초화+개발자 미리보기 (#631)
- 8초 타이밍 재조정 (#632), 중앙 점 방사 + T로 수렴(RadialGradient) (#633)

### 2-10. 알림
- 개발자 모드 로컬 알림 테스트 (#600), 알림 응답 파싱 강화 (#571)
- 서버 알림 → 기기 배너 브리지(폴러 시작 + 설정 연동 + 권한) (#638)

### 2-11. 회원 / 설정 / 인증
- 프로필 사진 변경(서버 DB 저장 + 업로드/스트림) (#605)
- 화면 테마 라이트/다크/시스템 3택 (#602), 상태바 아이콘 색 정합 (#622)
- 카카오 로그인에 이전 세션 토큰을 붙이지 않도록 수정 (#628)
- 관리자 로그인 시도 제한 + refresh 회전/무효화(Redis) (#550), 단일 JWT 보안 체인 (#549)

## 3. 품질 게이트

모든 PR은 다음 CI를 통과했다.
- Flutter Analyze & Test (앱), qtai-server Build & Test (백엔드)
- Requirements Guard(v3.1), Gitleaks, OpenAPI Spectral Lint, Branch/PR 규칙, claude-review

추가로 변경 범위에 맞춰 단위/위젯 테스트를 갱신했고, 인증 상태로 전체 앱을 띄우는 위젯 테스트까지
포함해 전체 그린을 유지했다.

## 4. 알려진 한계 / 후속 과제

- **알림**: FCM 미도입으로 앱 포그라운드 동안만 기기 배너 보장. 백그라운드 푸시는 FCM 도입 시 별도.
- **QT 영상**: 경로/플레이어/버튼은 정상. 실제 본문 영상 콘텐츠 제작·시딩은 별도 작업(현재 로컬은 샘플).
- **TTS 캐시**: 무료 호스팅 콜드스타트 영향을 줄이는 지연 캐시. 사전 생성 배치 연동은 후속.
- **로컬 시드 데이터**: 컨테이너/DB 초기화 시 사라지므로 스크립트 재실행 필요.
