## 같은 sprint 후속 처리 보장

이번 PR은 시스템 배치 AI 생성 job 트리거의 HTTP 접수 경로와 컨트롤러 단위 authority 검증까지만 포함합니다. `/api/v1/system/**` 전역 보안 게이트와 service account 기반 서버 간 인증은 이번 PR의 제외 범위이며, 같은 sprint 내 후속 PR에서 처리합니다.

- 후속 1: `service_accounts` 기반 service account token 검증 필터 및 `/api/v1/system/**` 전역 보안 설정 추가
- 후속 2: API 명세 기준 `promptVersionId`와 현재 UseCase 기준 `promptVersion` 문자열의 정합화 방안 Lead 합의 후 `ai_prompt_versions` 조회/매핑 구현

후속 PR이 merge되기 전까지 이 API는 `SYSTEM_BATCH`/`ROLE_SYSTEM_BATCH` authority를 컨트롤러에서 확인하는 최소 방어선으로 동작합니다.
