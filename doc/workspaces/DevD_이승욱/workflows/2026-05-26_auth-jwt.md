# 2026-05-26 인증 백엔드 및 Flutter 연동 (JWT + Kakao OAuth)

## 목표
카카오 OAuth 로그인부터 JWT 발급/갱신/무효화까지 인증 전체 흐름을 서버에 구현하고, Flutter 앱에서 카카오 SDK → 서버 인증 → 토큰 관리 → 자동 재인증까지 연동한다.

## 작업 내용
1. **서버 API 6개** — POST /auth/kakao(카카오 로그인+자동가입), POST /auth/token/refresh(토큰 갱신 rotation), POST /auth/logout, GET /members/nickname/check, GET /me/profile, PATCH /me/profile, POST /me/withdraw
2. **서버 신규 12파일** — AuthController, AuthService, KakaoOAuthClient, KakaoUserInfo, RefreshTokenStore(Redis), MemberAuthProvider(Entity), V6__create_member_auth_providers.sql 등
3. **서버 기존 7파일 구현 완료** — Member(프로필수정/탈퇴/닉네임잠금), MemberService, SecurityConfig(Stateless+JWT필터), ErrorCode 추가
4. **Flutter 인증 연동 7파일 신규** — SecureStorage, KakaoAuthService, AuthRepository, AuthState(sealed), AuthProvider(StateNotifier), SplashScreen, LoginScreen
5. **Flutter 기존 4파일 수정** — main.dart(카카오SDK초기화), AuthInterceptor(401 자동갱신 single-flight), ApiClient, AppRouter
6. **Security 정책** — JWT RS256 비대칭키, 환경변수 주입(기본값 없음), 카카오 access token 1회성 검증 후 폐기
